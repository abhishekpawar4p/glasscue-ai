# GlassCue AI

GlassCue AI is a full-stack real-time AI assistant for live meetings, presentations, technical discussions, and language speaking practice.

The application allows users to upload source materials, start a live session, receive one ready-to-say AI-generated response during the session, and review detailed feedback after the session.

## Project Goal

The goal of this project is to build a professional full-stack AI application using React, Node.js, Express.js, PostgreSQL, Prisma, and AI API integration.

## Core Features

- User authentication
- Source document upload
- Live session creation
- Real-time speech-to-text
- One final AI-generated answer during live mode
- Source-aware AI responses
- Session transcript storage
- Post-session review and feedback
- Language speaking practice mode
- Modern glass-style UI

## Tech Stack
